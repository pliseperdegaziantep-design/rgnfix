# RGN AI - Kolay Başlatma Paketi

Bu paket, RGN AI uygulamasını bilgisayarında çalıştırmak için hazırlanmıştır.

## Windows

1. ZIP dosyasını çıkar.
2. `RGN-AI-BASLAT-WINDOWS.bat` dosyasına çift tıkla.
3. Tarayıcıda `http://localhost:3000` açılır.

## Mac / Linux

Terminalde proje klasörüne girip çalıştır:

```bash
chmod +x RGN-AI-BASLAT-MAC-LINUX.sh
./RGN-AI-BASLAT-MAC-LINUX.sh
```

## Gerekli olan tek şey

Bilgisayarda Node.js LTS kurulu olmalı:

https://nodejs.org/

İlk çalıştırmada paketler indirileceği için internet gerekir. Sonraki çalıştırmalar daha hızlı açılır.

## Codex gerekli mi?

Hayır. Codex sadece kodu yapay zekâ ile geliştirmek için gerekir. Uygulamayı çalıştırmak için gerekli değildir.

## Gerçek kurulum dosyası istiyorsan

Windows için `.exe`, Android için `.apk`, Mac için `.dmg` üretilebilir; bunun için ilgili platformda build alınması gerekir.
